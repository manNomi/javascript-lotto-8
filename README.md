# javascript-lotto-precourse

# 로또를 만들자

1-1. 로또 발행

(구입 금액 / 1,000) 만큼 로또를 발행한다.

로또는 MissionUtils.Random.pickUniqueNumbersInRange(1, 45, 6)를 통해 6개 숫자를 생성

생성된 6개 숫자로 Lotto 클래스 인스턴스를 생성한다. ( Lotto 클래스의 number는 6개)

1-1. 당첨 확인

발행된 모든 로또를 당첨 번호, 보너스 번호와 비교한다.

각 로또의 당첨 등수(1~5등, 꽝)를 판별한다.

1-3. 결과 집계

총 당첨금을 계산한다.

(총 당첨금 / 구입 금액) \* 100을 하여 수익률을 계산한다.

2-1.로또 구매 금액 입력

- [] 숫자가 아닌 경우

- [] 1000원 단위가 아닌 경우

- [] 0원 이하인 경우

2-2.당첨 번호 입력

- [] 6개가 아닌 경우

- [] 숫자가 아닌 값이 포함된 경우

- [] 중복된 숫자가 있는 경우

- [] 1~45 범위를 벗어난 숫자가 있는 경우

- [] 공백이 포함되는 경우 (EX: 1, ,3)

2-3. 보너스 번호 입력

- [] 숫자가 아닌 경우

- [] 1~45 범위를 벗어난 숫자인 경우

- [] 당첨번호와 증복되는 경우

- 예상 출력

```

구입금액을 입력해 주세요.
8000

8개를 구매했습니다.
[8, 21, 23, 41, 42, 43]
[3, 5, 11, 16, 32, 38]
[7, 11, 16, 35, 36, 44]
[1, 8, 11, 31, 41, 42]
[13, 14, 16, 38, 42, 45]
[7, 11, 30, 40, 42, 43]
[2, 13, 22, 32, 38, 45]
[1, 3, 5, 14, 22, 45]

당첨 번호를 입력해 주세요.
1,2,3,4,5,6

보너스 번호를 입력해 주세요.
7
3개 일치 (5,000원) - 1개
4개 일치 (50,000원) - 0개
5개 일치 (1,500,000원) - 0개
5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
6개 일치 (2,000,000,000원) - 0개
총 수익률은 62.5%입니다.
```
