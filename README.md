# javascript-lotto-precourse

# 로또를 만들자

1-1. 로또 발행

(구입 금액 / 1,000) 만큼 로또를 발행한다.

로또는 MissionUtils.Random.pickUniqueNumbersInRange(1, 45, 6)를 통해 6개 숫자를 생성

생성된 6개 숫자로 Lotto 클래스 인스턴스를 생성한다. ( Lotto 클래스의 number는 6개)

1-1. 당첨 확인

발행된 모든 로또를 당첨 번호, 보너스 번호와 비교한다.

각 로또의 당첨 등수(1~5등, 꽝)를 판별한다.

1-3. 결과 집계

총 당첨금을 계산한다.

(총 당첨금 / 구입 금액) \* 100을 하여 수익률을 계산한다.

2-1.로또 구매 금액 입력

- [] 숫자가 아닌 경우

- [] 1000원 단위가 아닌 경우

- [] 0원 이하인 경우

2-2.당첨 번호 입력

- [] 6개가 아닌 경우

- [] 숫자가 아닌 값이 포함된 경우

- [] 중복된 숫자가 있는 경우

- [] 1~45 범위를 벗어난 숫자가 있는 경우

- [] 공백이 포함되는 경우 (EX: 1, ,3)

2-3. 보너스 번호 입력

- [] 숫자가 아닌 경우

- [] 1~45 범위를 벗어난 숫자인 경우

- [] 당첨번호와 증복되는 경우

3. FLOW

app(controller)

- [] 1. 로또 구매 금액 입력
- [] 2. 로또 구매 검증
- [] 3. lottoGame.buyLottos(구입금액)
- [] 4. 당첨 번호 입력
- [] 5. 당첨번호 parsing
- [] 6. 당첨번호 validate
- [] 7. 추가 당첨 번호 입력
- [] 8. 추가 당첨 번호 검증
- [] 9. result=lottoGame.calculateResult([당첨번호,추가 당첨])
- [] 10. result 출력
